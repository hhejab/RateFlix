# RateFlix – Movie Rating & Review System

RateFlix is a web-based movie rating and review system developed using PHP and MySQL for the IT8415 Database Programming 2 group project.

## Features

* User registration and login
* Role-based access (Admin, Creator, Viewer)
* Add and manage movies
* Upload movie posters
* Rate and comment on movies
* Search and filter movies
* Admin dashboard for managing content

## Technologies Used

* PHP
* MySQL
* HTML/CSS
* XAMPP
* phpMyAdmin

## How to Run

1. Start Apache and MySQL in XAMPP.
2. Import `sql/rateflix_db.sql`.
3. Import `sql/sample_data.sql`.
4. Place the project folder inside `htdocs`.
5. Open the following URL in your browser:

`http://localhost/RateFlix`
